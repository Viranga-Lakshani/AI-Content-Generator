Thank you for your interest in contributing! This file explains the friendly, human-centered way to help.

How to contribute
- Found a bug? Open an issue with "What I did", "What I expected", and "What happened".
- Want to add a feature? Open an issue to discuss the idea before you implement it.
- Ready to submit code? Open a pull request with a clear description of why the change helps and how you tested it.

Small gestures matter
- Keep language clear and polite in issues and PRs.
- Add tests for behavior changes.
- If you change prompts, explain the intended tone and audience.

Local development
1. Install dependencies: npm install
2. Run tests: npm run test
3. Lint: npm run lint
